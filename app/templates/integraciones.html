🧑‍💻 Cómo usar los skills con tu IA favorita
1
Obtén el mapa de skills (para dárselo a la IA)
Este endpoint devuelve la definición completa de todos los skills en formato JSON. Copia ese JSON y pégalo en la configuración de tu agente de IA.
URL 
{{ API_URL }}/api/v1/skills
🗺️ Ver skills JSON
2
Con OpenAI / GPT-4o — agrega los skills como tools
Copia y pega este código en tu proyecto Python. Los skills se cargan automáticamente del servidor:
Python — OpenAI 
import openai, requests, json

# 1. Cargar los skills desde el servidor
skills = requests.get("{{ API_URL }}/api/v1/skills").json()["data"]["skills"]

# 2. Convertirlos al formato que entiende OpenAI
tools = [{"type": "function", "function": {
    "name":        s["name"],
    "description": s["description"],
    "parameters":  s["input_schema"]
}} for s in skills]

# 3. Usar en tu chat
client = openai.OpenAI(api_key="TU_API_KEY")
response = client.chat.completions.create(
    model="gpt-4o",
    messages=[{"role": "user", "content": "Analiza https://example.com"}],
    tools=tools,
    tool_choice="auto"
)

# 4. Si el modelo llama un skill, ejecutarlo
msg = response.choices[0].message
if msg.tool_calls:
    for call in msg.tool_calls:
        args = json.loads(call.function.arguments)
        skill_name = call.function.name  # p.ej. "skill_analyze_url"
        
        # Llama al endpoint correspondiente
        result = requests.post(
            f"{{ API_URL }}/api/v1/analyze/url",
            json=args
        ).json()
        print(result["message"])
3
Con Google Gemini — Function Calling
Python — Gemini 
import google.generativeai as genai, requests

genai.configure(api_key="TU_API_KEY")

# Definir el skill directamente
analyze_code_fn = genai.types.FunctionDeclaration(
    name="skill_analyze_code",
    description="Analiza código fuente buscando vulnerabilidades OWASP Top 10",
    parameters={
        "type": "object",
        "properties": {
            "code":     {"type": "string", "description": "El código a analizar"},
            "language": {"type": "string", "description": "Lenguaje (python, js, java...)"}
        },
        "required": ["code"]
    }
)

model = genai.GenerativeModel("gemini-2.0-flash", tools=[analyze_code_fn])
response = model.generate_content("Analiza: password = '12345'")

# Si Gemini llama el skill, ejecutar la petición
if response.candidates[0].content.parts[0].function_call:
    fc = response.candidates[0].content.parts[0].function_call
    result = requests.post(
        "{{ API_URL }}/api/v1/analyze/code",
        json=dict(fc.args)
    ).json()
    print(result["message"])

Python
import requests

# Analizar código
respuesta = requests.post(
    "http://localhost:8001/api/v1/analyze/code",
    json={"code": "password = '12345'\neval(user_input)"},
    headers={"X-API-Key": "demo-token-12345"}  # opcional
)

datos = respuesta.json()
print(f"Score de seguridad: {datos['data']['score']}/100")
print(f"Hallazgos encontrados: {datos['data']['findings_count']}")
print(f"Resumen: {datos['message']}")

# Ver cada hallazgo
for h in datos['data']['findings']:
    print(f"  [{h['severity'].upper()}] {h['rule_id']}: {h['title']}")
    print(f"    Solución: {h['remediation'][:80]}...")
