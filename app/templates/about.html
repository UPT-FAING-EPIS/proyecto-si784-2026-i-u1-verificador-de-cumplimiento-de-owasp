Información del Proyecto

Verificador de Cumplimiento OWASP

Sistema inteligente de análisis automático de seguridad

🎯 Objetivo del Sistema

El Verificador de Cumplimiento OWASP es una herramienta educativa avanzada diseñada para analizar código fuente y URLs en busca de vulnerabilidades de seguridad basadas en los estándares OWASP Top 10. Proporciona a desarrolladores y estudiantes una forma automática y eficiente de identificar problemas de seguridad comunes y recibir recomendaciones específicas para remediarlos.

🔬 Análisis Automático

Motor de análisis que utiliza patrones regex y reglas heurísticas para detectar vulnerabilidades comunes en código Python, JavaScript y otros lenguajes de programación.

📊 Puntuación Inteligente

Sistema de scoring basado en severidad que calcula una puntuación de 0-100. Las vulnerabilidades de alta severidad deducen más puntos que las de baja severidad.

✅ Soluciones Paso a Paso

Cada hallazgo incluye guías prácticas detalladas que explican cómo remediar el problema, con ejemplos de código seguro.

📚 Wiki Educativa

Documentación completa sobre cada tipo de vulnerabilidad, incluyendo explicaciones del riesgo, técnicas de explotación y prácticas seguras.

📡 Monitoreo en Tiempo Real

Sistema de logging que registra todos los accesos al sistema, mostrando IP, ruta y timestamp para auditoría y análisis de uso.

🔐 URL Scanning

Análisis de URLs activas que verifica la presencia de cabeceras HTTP de seguridad críticas y expone información de infraestructura.

🚀 Características Principales

Detección de Vulnerabilidades

  • OWASP-A02: Exposición de Secretos - Detecta contraseñas, API keys y tokens hardcodeados
  • OWASP-A03: Funciones Peligrosas - Identifica eval(), exec(), pickle.loads() y subprocess inseguros
  • OWASP-A04: Validación Insuficiente - Verifica manejo inseguro de entrada de usuario
  • OWASP-A05: Headers Faltantes - Comprueba cabeceras de seguridad HTTP críticas
  • OWASP-A06: Divulgación de Info - Expone headers del servidor que revelan tecnología usada

Dashboard y Reportes

  • Panel de Control - Estadísticas en tiempo real (total análisis, promedio score, hallazgos críticos)
  • Reportes Detallados - Información completa por análisis con problema, evidencia y solución
  • Historial de Escaneos - Acceso a todos los análisis realizados anteriormente
  • Monitoreo de Accesos - Log de quién accede al sistema, cuándo y desde dónde

Interfaz de Usuario

  • Diseño Moderno - Interfaz oscura profesional con gradientes y efectos visuales
  • Responsive - Compatible con dispositivos móviles, tablets y desktops
  • Accesibilidad - Colores y contraste diseñados para máxima legibilidad

🏗️ Arquitectura Técnica

Backend: FastAPI con Python 3.11+

Frontend: Jinja2 Templates + CSS3 Moderno

Almacenamiento: In-Memory Store (sin base de datos externa)

Análisis: Regex-based pattern matching + heurísticas de seguridad

📊 Métricas y Evaluación

El sistema utiliza un algoritmo de puntuación transparente que calcula una puntuación de 0 a 100 basada en:

  • Severidad: Vulnerabilidades HIGH (-30 pts), MEDIUM (-15 pts), LOW (-5 pts)
  • Acumulativo: La puntuación desciende hasta un mínimo de 0
  • Interpretación: 90-100 = Excelente | 70-89 = Aceptable | 50-69 = Necesita mejora | <50 = Crítico
