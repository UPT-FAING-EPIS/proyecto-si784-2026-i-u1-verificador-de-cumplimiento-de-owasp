Capacidades del Sistema de Escaneo

OWASP Verificator analiza estáticamente archivos de código local, URLs de aplicaciones activas y repositorios Git públicos para auditar el cumplimiento del OWASP Top 10.

🔑 Detección de Secretos

Identifica contraseñas hardcodeadas, tokens de acceso y llaves de API expuestas.

⚠️ Funciones Peligrosas

Busca llamadas inseguras a eval(), exec() y comandos del sistema (RCE).

📝 Inyecciones e Input

Inspecciona variables de entrada directa propensas a SQLi, XSS y Path Traversal.

🔒 Headers de Seguridad

Inspecciona configuraciones de seguridad HTTP (X-Frame-Options, CSP, HSTS).

📊 Puntuación Inteligente

Genera un Score del 0 al 100 ponderando penalidades por vulnerabilidad detectada.

🎯 Guía de Remediación
