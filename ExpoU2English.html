What is OWASP?

OWASP stands for Open Web Application Security Project. It is an international non-profit organization that works to make software and websites more secure.

Key Facts

🌍 Founded 2001
🏛️ Type Non-profit organization (free for everyone)
🎯 Goal Help people build safer websites and apps
👥 Community Thousands of experts around the world

What is the OWASP Top 10?

It is a list of the 10 most dangerous security problems found in websites. Think of it as a "checklist" that helps developers know what to fix first.

💡 Our project checks websites and code against this list automatically, so developers don't have to do it by hand.

The 10 Most Common Security Risks

A01🔓Broken Access Control
A02🔐Weak Encryption
A03💉Code Injection
A04🏗️Insecure Design
A05⚙️Bad Configuration
A06📦Outdated Software
A07🪪Login Problems
A08🧬Data Tampering
A09📝No Activity Logs
A10🌐Fake Requests

Slide 4

Problem Definition

Every year, millions of websites get hacked because they have security problems that nobody checked for.

Why Does This Happen?

🔴 Developers don't know about security

Most programmers are never taught how to protect their websites

🔴 No free tools available

Professional security tools cost thousands of dollars per year

🔴 Checking by hand is too slow

Manually reviewing code for problems takes days or even weeks

🔴 Students have no practice

Universities don't provide simple tools to learn about cybersecurity

What Can Go Wrong?

  • 📉 Personal data gets stolen — names, passwords, credit cards
  • 🚫 Hackers get access — they can control the entire system
  • ⚠️ Websites crash — users can't access the service
  • 💸 Companies lose money — lawsuits, fines, lost customers
"There is no simple, free tool for students to quickly check if their websites are safe."

That's exactly what our project solves.
